# Cosmodrome
A cross-platform subsonic (specifically navidrome) music client built within Flutter.
<hr >

## Downloads

As of right now, there are no downloads, you can go and compile this yourself if you would like an install.

## Platform Support
Current support for the client is as following:

 - [X] Linux
 - [X] iOS
 - [X] Android
 - [ ] Windows (unknown)
 - [ ] Web (unknown)
 - [ ] MacOS (unknown)
